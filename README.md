# 📊 Dashboard TSL - B&B Hotels / Transgourmet

Dashboard interactif pour la visualisation des indicateurs de qualité Transgourmet (TSL) pour B&B Hotels.

## 🎯 Fonctionnalités

### 5 Onglets Principaux:

1. **📈 Vue Globale**
   - KPIs globaux (sites, taux service, lignes, incidents, clients, produits)
   - Graphiques de distribution des motifs
   - Top produits par incidents

2. **🏨 Par Site**
   - Tableau détaillé des 15 sites Transgourmet
   - Comparaison des performances
   - Identification des meilleurs/pires sites

3. **📦 Produits**
   - Liste des 30+ produits
   - Filtrage par catégorie (Frais, Épicerie, Ambiant)
   - Incidents par référence

4. **👥 Clients/Hôtels**
   - 50+ hôtels B&B
   - Recherche par nom/ville
   - Taux de service détaillé

5. **⚠️ Réclamations**
   - 8 motifs principaux
   - Distribution en pourcentage
   - Analyse des incidents

## 📊 Données Incluses

- **Période**: Mars 2026 (01/03/26 - 31/03/26)
- **Sites**: 15 centres Transgourmet
- **Clients**: 50+ hôtels B&B Hotels
- **Produits**: 30+ références
- **Motifs**: 8 types de réclamation

## 🚀 Utilisation

### Option 1: Ouverture Directe
```bash

```
